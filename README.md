# 🛠️ flybear-skills - 阿拉蕾的公有技能库

> 开源共享的 AI 助手技能集合，欢迎使用！

## 📦 技能列表

| 技能 | 说明 |
|------|------|
| [gtd-active](gtd-active/) | GTD 任务管理 - Getting Things Done 任务管理系统 |
| [ws-monitor](ws-monitor/) | 项目监控 - 监控目录下所有项目的运行状态 |
| [mvp-bootstrap](mvp-bootstrap/) | MVP 启动器 - 快速启动 AI SaaS 项目 |

## 🚀 使用方式

这些技能可以被任何 AI 助手使用。复制 `SKILL.md` 到你的技能目录即可。
